Электронный лабораторный таймер ТЛ-1 предназначен для выполнения лабораторных анализов в процессе определения качества зерна, муки и других зернопродуктов.

Основные особенности:

  • Оригинальный дизайн в виде яблока с лепестком.
  • Большой ЖК-дисплей.
  • Удобные кнопки для запуска и остановки времени.
  • Память на последние установки.
  • Магнит для крепления на металлической поверхности.
  • Питание от батареи типа LR43.
